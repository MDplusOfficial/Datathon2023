Problem 2

You want to know whether there is a difference between the age of those who are alive and those who are dead. Choose the correct statistical test and understand why. Is the result statistically significant?

#age of alive
hist(data %>% filter(dead == 0) %>% pull(age))

#age of dead
hist(data %>% filter(dead == 1) %>% pull(age))

wilcox.test(age ~ dead, data = data, alternative = "two.sided")

Because age doesn’t appear to be normally distributed, one of the technical assumptions for using the t-test is violated. Thus, we use the non-parametric test, Wilcox rank-sum test, instead to test our null hypothesis of there being no difference between the age of those who are alive and those who are dead. We fail to reject the null hypothesis (p-value = 0.169).

Problem 3

You’re trying to assess whether you can build an explanatory model between the association of tumor size and death. What model should you use?

Logistic, death is a binary outcome.

Adjust for age, number of nodes, estrogen receptor status, and progesterone receptor status. Is the association statistically significant?

library(broom)
my_model <- glm(dead ~ tumor_size + age + n_nodes + er + pr,
                family = binomial,
                data = data)
broom::tidy(my_model,
            exponentiate = TRUE,
            conf.int = TRUE)

Bonus: What different type of model could you run given the variables you were provided? (Hint: look at the variable follow_up)

Cox Proportional Hazards, where the measure of association is now something called a hazard ratio. This is part of a field of statistics called survival analysis.

library(survival)
my_model <- coxph(Surv(follow_up, dead) ~ tumor_size + age + n_nodes + er + pr,
                data = data)
