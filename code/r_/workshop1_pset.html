Quick Problem 1

Import the toy dataset (“workshop1_toydata”) provided to you. You’re going to want to install the package readxl and use the readxl::read_excel function. If you just immediately imported the dataset, you’ll notice that it included the header “Mount Sinai Research Group”, which you don’t want. The reason it’s there is that more often than not, your medical school PI or research group will give you untidy data, which has a bunch of gunk in it. Run ?read_excel to pull up the help and see if you can find an argument that will let you skip the first row.

If you’re really having trouble, you can always import the dataset manually by going to File -> “Import Dataset” -> selecting your file type. It has a nice interface that can walk you through data import.

FYI: There’s a lot of other things you can toggle too!

Problem 2

First, run:

#First run install.packages("medicaldata") in your console
library(medicaldata)

We will be using the covid_testing dataset. You can get this object into your global environment by running df <- medicaldata::covid_testing . This problem will require you to use the “Data Wrangling Verbs You Should Know” as well as some additional verbs from the Data Wrangling Cheatsheet. This problem mimcs what you’d be doing in a clinical research study. Note there is always more than one code solution. Code generally differs by personal preference, speed, and readability.

  1. How many observations are in this dataset? How many unique patients are in this dataset (unique is defined as a subject_id appearing only once)? Note: for the remaining questions, uniqueness is implicit. If you’re working on cohort data, you should consider unique patients.

  2. How many males and females are in this dataset?

  3. Find the average age of females who got tested at a clinical lab.

  4. Find the median age of males who got tested between the 7th and 14th days (pan_day).

  5. How many inpatient and emergency patients (use patient_class) tested positive?

  6. There is a data entry error in age. Find it. Hint: what ages would be implausible?

  7. Create a new binary variable called medicare_eligible using age. Assign it to an object called df2. The variable is defined as age greater than or equal to 65. You can use either ifelse() or case_when().

  8. Which patient has the highest combined col_rec_tat and rec_ver_tat?
