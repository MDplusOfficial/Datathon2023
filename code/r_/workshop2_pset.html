For this problem set, you will be using randomly sampled data from a real-life cancer cohort.

Data Dictionary
Variable Name Description
age Age at which participant was diagnosed with breast cancer
follow_up Time from diagnosis to death/last follow-up
dead 0 = alive, 1 = dead
tumor_size Size of largest tumor (mm)
n_nodes Number of positive lymph nodes at diagnosis
tumor_grade Tumor grade (1 = not terrible, 2 = moderately terrible, 3 = terrible)
er Estrogen receptor status (0 = negative, 1 = positive)
pr Progesterone receptor status (0 = negative, 1 positive)

Problem 2

You want to know whether there is a difference between the age of those who are alive and those who are dead. Choose the correct statistical test and understand why. Is the result statistically significant?

Problem 3

You’re trying to assess whether you can build an explanatory model between the association of tumor size and death. What model should you use?

Adjust for age, number of nodes, estrogen receptor status, and progesterone receptor status. Is the association statistically significant?

Bonus: What different type of model could you run given the variables you were provided? (Hint: look at the variable follow_up)
